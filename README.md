# NGO Connect 🌱

A MERN Stack NGO Volunteer Management System that helps manage plantation events, volunteers, and environmental initiatives.

## Features

- User Registration & Login
- JWT Authentication
- Volunteer Dashboard
- NGO Event Management (Add, Edit, Delete Events)
- Profile Management
- Responsive UI with Bootstrap
- MongoDB Atlas Integration

## Tech Stack

### Frontend
- React.js
- React Router
- Axios
- Bootstrap

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication

## API Endpoints

### Authentication

- POST `/api/register`
- POST `/api/login`

### Events

- GET `/api/events`
- POST `/api/events`
- PUT `/api/events/:id`
- DELETE `/api/events/:id`
